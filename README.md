# User Management API

A REST API built with FastAPI and Python to manage user data.

## Tech Stack

Python 3.12
FastAPI
Uvicorn
Pydantic
Git & GitHub

## Purpose

This project demonstrates real-world backend API development using FastAPI, clean project setup, and proper Git workflow as part of Python backend development.

## Features

Create User
List Users
Automatic API documentation (Swagger)

## Run the Project

Create virtual environment:

py -m venv venv

Activate:

PowerShell

.\venv\Scripts\Activate.ps1

CMD

venv\Scripts\activate.bat

Install dependencies:

pip install fastapi uvicorn

Run server:

py -m uvicorn app:app --reload

Open in browser:

http://127.0.0.1:8000
http://127.0.0.1:8000/docs
